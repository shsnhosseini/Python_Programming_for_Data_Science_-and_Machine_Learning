# Python Programming for Data Science and Machine Learning with Telecom and IoT Examples

A learning repository covering Data science libraries, with practical examples and datasets for hands-on practice.

## 📚 Project Contents

- **Pandas.ipynb** - Data manipulation and analysis:
  - DataFrames and Series
  - Data cleaning and transformation
  - Data aggregation and grouping

- **Matplotlib.ipynb** - Data visualization:
  - Creating plots and charts
  - Customizing visualizations
  - Multi-panel figures

### Datasets

- **Electric_Vehicle_Population_Data.csv** - Real-world EV registration data
- **iot_data.csv** & **iot_data_2.csv** - IoT sensor measurements for analysis practice
